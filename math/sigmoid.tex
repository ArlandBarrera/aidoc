This function takes any real-value input and maps it to an output between 0 and 1, creating a charasterisic S-shaped curve. It is also known as \textbf{logistic curve}.

\begin{listequbox}
  {f(x) = \dfrac{1}{1+e^{-x}}}{equsigmoid}{Sigmoid function}
\end{listequbox}

The S-shaped curve of the function is shown below.

\begin{grafica}
\center
\begin{tikzpicture}
  \begin{axis}[
    xmin=-6.3,xmax=6.3,
    ymin=0,ymax=1.1,
    ytick={0,0.5,1},yticklabels={$0$,$0.5$,$1$},
    axis lines=center
    ]
    \addplot[color=blue,samples=100,domain=-6:6,thick]{1/(1+exp(-x)};
  \end{axis}
\end{tikzpicture}
\caption{Sigmoid function}
\end{grafica}

\textbf{Charasterisics:}

\begin{itemize}
  \item Domain: $(-\infty,\infty)$
  \item Range: $(0,+1)$
  \item Intersection: $f(0)=0.5$
\end{itemize}
